# 🎮 买块联机工具

我的世界 IPv6 联机助手 - 帮助玩家快速测试连通性、获取地址并配置防火墙，让联机更简单！

## ✨ 功能特点

### 🔍 连通性测试（核心功能）
- 双方交换 IPv6 地址后，一键测试连通性
- 自动检测运营商（电信/移动/联通）
- 智能判断谁适合当房主
- 提前发现跨运营商路由问题

### 📋 IPv6 地址管理
- 自动扫描本机所有 IPv6 地址
- ⭐ 智能推荐临时地址（更安全）
- 一键复制地址，格式化为 `[地址]:` 方便粘贴
- 区分可用/不可用地址

### 🔥 防火墙配置（房主专用）
- 一键配置 Windows 防火墙入站规则
- 自动创建/更新/删除规则
- 支持任意端口设置

### 🌐 其他功能
- 快速测试 IPv6 连通性
- 美观的启动画面和现代化 UI

## 🎯 使用说明

### 完整流程（推荐）

**第一步：双方都打开工具**
- 房主和玩家都需要下载并运行工具
- 以管理员身份运行

**第二步：交换 IPv6 地址**
- 在地址列表中找到带 ⭐推荐 标签的地址
- 点击"复制"按钮
- 通过 QQ/微信 等发给对方

**第三步：连通性测试**
- 点击"🔍 连通性测试"按钮
- 输入对方的 IPv6 地址
- 查看测试结果，确定谁当房主

**第四步：房主设置端口**
- 由测试结果推荐的一方当房主
- 房主打开游戏，创建房间/开启局域网
- 记下游戏显示的端口号（如 25565）
- 在工具中输入端口号，点击"设置端口"

**第五步：玩家加入**
- 玩家在游戏中输入：`[房主IPv6地址]:端口`
- 例如：`[2409:890d:380:18a3:5ad:de6b:8552:ff6a]:25565`

**第六步：结束后删除规则**
- 联机结束后，房主点击"删除规则"关闭端口
- 确保网络安全

### 常见问题

**Q: 为什么需要连通性测试？**
A: 跨运营商（如电信连移动）可能出现单向不通的情况，提前测试可以避免进游戏后发现连不上。

**Q: Ping 通就一定能连上游戏吗？**
A: 不一定 100%，但成功率很高（95%+）。还需要确保房主正确设置了防火墙端口。

**Q: 测试显示连不上怎么办？**
A: 
1. 尝试让对方当房主（反向连接）
2. 使用 VPN 组网工具（如 Zerotier、Radmin VPN）
3. 等几天再试（可能是运营商临时路由问题）

**Q: 为什么推荐临时地址？**
A: 临时地址会定期更换，相对更安全，不容易被追踪。
